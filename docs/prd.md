# PRD

次の項目を選択できるリンクメニューがあり、それぞれのページに遷移することができる
- overviewページ
    - 各種プラットフォームへのリンクボタン
    - 私の全体概要と写真
- about meページ
    - 私の経歴についての説明ページ
- productページ
    - 私が過去に作ったプロダクトの紹介ページ
- blog listページ
    - 私が書いたブログの一覧ページ

productページとblog listページについては新しいものを追加するのが容易であることが絶対条件

blogの内容やproductの内容は私が手動で後で追加する
アイコン画像も私があとで手動で追加する
